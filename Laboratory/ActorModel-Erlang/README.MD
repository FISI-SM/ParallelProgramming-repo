## üìò 2. Uso de listas y `foldl`

**Archivo:** `lista.erl`  
Suma los elementos de una lista usando `lists:foldl`.

```erlang

```

**Ejemplo:**


## üìò 4. Paso de mensajes con respuesta

**Archivo:** `pingpong.erl`  
Simula un ping-pong entre procesos.

```erlang
pong() ->
    receive
        {ping, From} -> From ! pong
    end.
```

**Ejemplo:**

```erlang
1> pingpong:ping().

```

## üìò 5. Uso de `spawn` con argumentos
